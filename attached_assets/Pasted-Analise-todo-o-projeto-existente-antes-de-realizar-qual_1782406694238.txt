ETAPA 1 — Ajustar modelo de imóveis

Atualmente existe o conceito de codigoExterno.

Remover a dependência conceitual de codigoExterno como identificador principal.

Criar um novo campo:

identificadorOrigem

Tipo:
string

Descrição:

Identificador único fornecido pelo conector responsável pela importação.

Exemplos:

Helena:
identificadorOrigem = código do imóvel

Casa Linhares:
identificadorOrigem = URL do imóvel

Outras fontes:
identificadorOrigem = qualquer identificador estável definido pelo conector

Este campo será utilizado futuramente para evitar duplicidades.
