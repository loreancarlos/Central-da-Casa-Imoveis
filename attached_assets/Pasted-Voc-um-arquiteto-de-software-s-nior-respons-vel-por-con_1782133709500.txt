Você é um arquiteto de software sênior responsável por construir a primeira versão de um sistema chamado "Central da Casa Property Finder".

# Objetivo do Produto

Este sistema é uma ferramenta interna para corretores de imóveis.

O principal objetivo é permitir que um corretor cadastre clientes compradores e encontre imóveis compatíveis em uma base unificada.

Neste MVP inicial NÃO implementar scraping ainda.

Os imóveis serão cadastrados inicialmente por seed/mock para validar a experiência.
